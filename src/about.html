Quick notes

Tips to remember while working on the project:

  1. Read the request into a pre-allocated buffer or resize a std::string before reading.
  2. Parse the first line for method and path.
  3. Set the correct Content-Type and Content-Length headers before sending a response.
  4. Handle automated browser requests like /favicon.ico to avoid confusion.
  5. Refactor when the single file grows beyond ~300â€“400 lines.
